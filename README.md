# Movie search app

This project allows a user to search for movies by name using themoviedb.org API. If results are found they will be displayed in a nice and readable format, if no results are found then a friendly message will be displayed that their search didnâ€™t find any results.

## To run the application

`npm install`
`npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Dependencies used
 
 In this application following tools are used:
 1. react [https://www.npmjs.com/package/react]
 2. react-loader-spinner [https://www.npmjs.com/package/react-loader-spinner]
 3. moment [https://www.npmjs.com/package/moment]

## `npm test`

Launches the test runner in the interactive watch mode.
